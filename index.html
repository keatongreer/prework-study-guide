HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML elements are used to build the structure of a webpage.
  • Meta elements define metadata such as the character set, description, keywords, author, and viewport.
  • When possible, it is important to use semantic HTML elements, because they make your HTML easier to read by other developers and the browser.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A class selector can be used to freely choose which elements to style using a CSS rule and HTML class declaration.
  • A box shadow property adds a shadow effect around the frame of an element given X and Y offsets, blur and spread radius, and a color.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • pull request: used to merge a comparison or feature branch into the base or main branch
  • When a new feature is being added to a web app, a feature branch is created from the main branch in order to keep the new development separate from the working version.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function can be used to easily reuse a section of code throughout a script.
  • A random integer can be generated by using Math.floor(Math.random() * range).
